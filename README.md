## 项目目标
搭建一个后端的服务器集群框架
> 运行在 Tomcat/Jetty/端口直连 上 ,对外使用Http协议提供服务
> 内部服务由集群提供 ,并注册在ZooKeeper上 ; 服务传输由Protobuf序列化
> 使用ORM 和 自动化的代码/文件生成 简化数据库的操作
> ...
